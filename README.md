The Vrinda Sales Report Dashboard is a comprehensive tool created using Microsoft Excel to analyze and visualize key sales metrics for Vrinda, a leading retail business. This dashboard provides actionable insights into sales performance, inventory levels, and customer behavior, enabling data-driven decision-making and strategic planning.

Key Features
Sales Performance Analysis: Tracks key sales metrics, including total sales, sales growth, and sales by product category, providing a clear view of business performance.
Inventory Management: Monitors inventory levels in real-time, helping to optimize stock levels and reduce overstock or stockouts.
Customer Insights: Analyzes customer behavior, including purchase patterns and customer segmentation, to identify opportunities for targeted marketing.
Interactive Visualizations: Utilizes PivotTables, charts, and other Excel features to create an interactive and user-friendly interface, allowing users to explore data from different perspectives.
Automated Updates: The dashboard is designed to update automatically with new data, ensuring that the insights remain relevant and timely.
Technology Stack
Tool: Microsoft Excel
Key Functions: VLOOKUP, INDEX-MATCH, PivotTables, Data Validation, Conditional Formatting
Data Source: Internal sales and inventory data from Vrinda
